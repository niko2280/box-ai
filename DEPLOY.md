# 📱 Как открыть на телефоне iQOO Neo 10

## ✅ Способ 1: По локальной сети (РЕКОМЕНДУЕТСЯ)

### На компьютере:
```bash
cd "/home/dronolyub/box ai"
npm run dev
```

Сервер запустится на: `http://192.168.0.109:5173`

### На телефоне:
1. Подключись к той же Wi-Fi сети
2. Открой Chrome или любой браузер
3. Введи адрес: **`http://192.168.0.109:5173`**
4. Готово! 🎉

---

## 🌐 Способ 2: Деплой на Vercel (постоянный доступ)

### 1. Установи Vercel CLI:
```bash
npm install -g vercel
```

### 2. Задеплой:
```bash
cd "/home/dronolyub/box ai"
vercel --prod
```

### 3. Получишь ссылку типа:
`https://shadowcoach-ai.vercel.app`

Открывай её на любом устройстве!

---

## 🚀 Способ 3: Через ngrok (временный публичный URL)

### 1. Установи ngrok:
```bash
# Скачай с https://ngrok.com/download
# Или через snap:
sudo snap install ngrok
```

### 2. Запусти туннель:
```bash
ngrok http 5173
```

### 3. Получишь URL:
```
Forwarding: https://abc123.ngrok.io -> http://localhost:5173
```

Открывай этот URL на телефоне!

---

## 📦 Способ 4: Собрать и хостить статику

### 1. Собери проект:
```bash
npm run build
```

### 2. Залей папку `dist/` на любой хостинг:
- GitHub Pages
- Netlify
- Firebase Hosting
- Cloudflare Pages

---

## 💡 Рекомендация для iQOO Neo 10:

**Используй Способ 1** для разработки и тестирования.

**Используй Способ 2 (Vercel)** для постоянного доступа — это бесплатно и быстро!

### Команда для быстрого деплоя:
```bash
npm install -g vercel
vercel --prod
```

После деплоя добавь сайт на главный экран:
1. Открой в Chrome
2. Меню → "Добавить на главный экран"
3. Теперь это PWA! 🚀

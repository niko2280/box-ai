# 🥊 ShadowCoach AI

Персональный ИИ-тренер по боксу, который анализирует бой с тенью через камеру в реальном времени.

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## ✨ Возможности

- 📹 **Анализ в реальном времени** — TensorFlow.js + MoveNet отслеживают каждый удар
- 🤖 **ИИ-разбор** — Groq API (llama-3.3-70b-versatile) даёт детальный фидбек
- 📊 **9 категорий оценки** — джеб, кросс, хук, апперкот, защита, футворк, стойка, скорость, комбинации
- 📈 **Прогресс** — графики, статистика, тепловая карта активности
- 📋 **Персональный план** — ИИ составляет план тренировок на неделю
- 💬 **Чат с тренером** — задавай вопросы ИИ-тренеру
- 🎯 **Без регистрации** — всё хранится локально в IndexedDB

## 🛠 Технологии

- React 18 + TypeScript
- TensorFlow.js + MoveNet (распознавание поз)
- Groq API (llama-3.3-70b-versatile)
- Tailwind CSS v4 + Framer Motion
- Recharts (графики)
- IndexedDB (хранение данных)
- Lucide React (иконки)

## 📱 Страницы

1. **Главная** — общий рейтинг, оценки по категориям, серия тренировок
2. **Тренировка** — запись с камеры, анализ в реальном времени, детальные результаты
3. **Прогресс** — графики, история тренировок, сравнение
4. **План** — ИИ-генерация плана на неделю
5. **Советы** — чат с ИИ-тренером, библиотека советов, видео

## ⚙️ Настройки

- Стойка: Orthodox / Southpaw
- Уровень: Новичок / Любитель / Продвинутый
- Groq API ключ (по умолчанию встроен)
- Модель: llama-3.3-70b-versatile / llama-3.1-8b-instant
- Звук, вибрация, показ скелета позы
